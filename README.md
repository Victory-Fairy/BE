# 승요의 일기장
야구 직관, 집관 기록 서비스

## 기술 스택
- 언어 & 프레임워크
  - Java 17
  - Spring Boot 3.2.4
  - Spring Batch
  - Spring Data JPA + QueryDSL

- 데이터베이스 & 캐시
  - MySQL
  - Redis

- 메시지 브로커 & 이벤트 처리
  - Redis Streams

- 운영 환경
  - Ubuntu
  - Docker
  - Docker Swarm

- 인증 & 보안
  - JWT
